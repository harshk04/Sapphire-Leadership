# Sapphire Leadership & Advisory (Next.js)

Pixel-perfect marketing site built from `screen.png` using Next.js (App Router), TypeScript, Tailwind CSS, Framer Motion, and Lucide icons.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Then open `http://localhost:3000`.

## Production build

```bash
npm run build
npm start
```

## Notes

- Image assets are currently derived from `screen.png` and stored in `public/images`.
- The consultation form is client-side only (no backend).
